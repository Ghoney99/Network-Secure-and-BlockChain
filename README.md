# 암호화

이 저장소는 다양한 암호화 기법을 구현한 Python 스크립트들을 포함하고 있습니다.

## 1. 대치 암호 (1.1_대치암호.py)

### 개요
이 프로그램은 대치 암호(Substitution Cipher)를 구현한 것입니다. 각 알파벳을 다른 알파벳으로 변환하여 텍스트를 암호화합니다.

### 기능
- 사용자로부터 평문을 입력받아 암호문으로 변환
- 생성된 암호문을 복호화하여 원래의 평문으로 복원

### 사용 방법
1. Python이 설치된 환경에서 `1.1_대치암호.py` 파일을 실행합니다.
2. 프로그램의 지시에 따라 평문을 입력합니다.
3. 암호문과 복호문이 출력됩니다.

### 주요 함수
- `Encryption(text)`: 주어진 평문을 암호화하여 암호문을 반환합니다.
- `Decryption(text)`: 주어진 암호문을 복호화하여 원래의 평문을 반환합니다.

### 요구 사항
- Python 3.x 이상
- random 모듈 (Python 기본 제공)

## 2. Vigenère 및 자동 키 암호 (1.2_Vigenere및Auto키암호.py)

### 개요
이 프로그램은 Vigenère 암호와 자동 키 암호(Autokey Cipher)를 구현한 것입니다.

### 기능
- Vigenère 암호화 및 복호화
- 자동 키 암호화 및 복호화

### 사용 방법
1. Python이 설치된 환경에서 `1.2_Vigenere및Auto키암호.py` 파일을 실행합니다.
2. 프로그램의 지시에 따라 평문과 키를 입력합니다.
3. 암호문과 복호문이 출력됩니다.

### 주요 함수
- `vigenere_E(text, key)`: Vigenère 암호화를 수행하고 암호문을 반환합니다.
- `vigenere_D(text, key)`: Vigenère 복호화를 수행하고 원래의 평문을 반환합니다.
- `autokey_E(text, key)`: 자동 키 암호화를 수행하고 암호문을 반환합니다.
- `autokey_D(text, key)`: 자동 키 복호화를 수행하고 원래의 평문을 반환합니다.

### 요구 사항
- Python 3.x 이상

## 3. AES 암호화 (1.3_AES암호화.py)

### 개요
이 프로그램은 AES(Advanced Encryption Standard) 방식으로 파일을 암호화하고 복호화하는 기능을 제공합니다.

### 기능
- 파일을 AES 방식으로 암호화
- 암호화된 파일을 복호화하여 원본 데이터를 출력

### 사용 방법
1. Python 및 cryptography 라이브러리가 설치된 환경에서 `1.3_AES암호화.py` 파일을 실행합니다.
2. 암호화할 데이터가 포함된 `data.txt` 파일을 준비합니다.
3. 프로그램이 실행되면, 암호화된 내용이 `encrypted.txt` 파일에 저장되고, 복호화된 원본 내용이 출력됩니다.

### 주요 함수
- `Fernet.generate_key()`: 새로운 암호화 키를 생성합니다.
- `fernet.encrypt(data)`: 주어진 데이터를 암호화합니다.
- `fernet.decrypt(encrypt_data)`: 암호화된 데이터를 복호화합니다.

### 요구 사항
- Python 3.x 이상
- cryptography 라이브러리 설치: `pip install cryptography`

## 4. RSA 암호화 (2.1_RSA암호화.py)

### 개요
이 프로그램은 RSA 공개 키 암호화와 AES 대칭 키 암호화를 결합하여 메시지를 안전하게 전송하는 기능을 제공합니다.

### 기능
- RSA 공개 키와 개인 키를 사용하여 AES 키를 암호화
- AES를 사용하여 평문 메시지를 암호화 및 복호화

### 사용 방법
1. Python 및 cryptography 라이브러리가 설치된 환경에서 `2.1_RSA암호화.py` 파일을 실행합니다.
2. RSA 공개 키(`public_key.pem`)와 개인 키(`private_key.pem`) 파일을 준비합니다.
3. 프로그램의 지시에 따라 평문 메시지를 입력합니다.
4. 암호화된 메시지와 복호화된 원본 메시지가 출력됩니다.

### 주요 함수
- `Fernet.generate_key()`: 새로운 AES 키를 생성합니다.
- `public_key.encrypt()`: RSA 공개 키로 AES 키를 암호화합니다.
- `private_key.decrypt()`: RSA 개인 키로 AES 키를 복호화합니다.
- `fernet.encrypt()`: 평문 메시지를 AES로 암호화합니다.
- `fernet.decrypt()`: 암호문을 AES로 복호화합니다.

### 요구 사항
- Python 3.x 이상
- cryptography 라이브러리 설치: `pip install cryptography`

## 5. 블록체인 개인키/공개키 생성 (2.2_블록체인_개인키_공개키생성.py)

### 개요
이 프로그램은 타원 곡선 암호화(ECC)를 사용하여 개인 키와 공개 키를 생성합니다.

### 기능
- 개인 키를 생성
- 생성된 개인 키를 기반으로 공개 키를 계산
- 개인 키와 공개 키를 16진수 및 10진수 형식으로 출력

### 사용 방법
1. Python 환경에서 `2.2_블록체인_개인키_공개키생성.py` 파일을 실행합니다.
2. 프로그램이 랜덤한 개인 키를 생성한 후, 해당 키에 대한 공개 키를 계산하여 출력합니다.

### 주요 함수
- `double_and_add(p, private_key, Gx, Gy)`: 개인 키를 사용하여 공개 키를 계산하는 함수입니다.
- `extended_euclidian(n, b)`: 두 수의 최대 공약수를 계산하고, 역원을 구하는 함수입니다.
- `point_add(p, x1, y1, x2, y2)`: 타원 곡선에서 두 점을 더하는 함수입니다.

### 요구 사항
- Python 3.x 이상

## 6. 전자서명 (3.1_전자서명.py)

### 개요
이 프로그램은 타원 곡선 암호화를 기반으로 한 전자 서명 생성 및 검증 기능을 제공합니다.

### 기능
- 개인 키 생성
- 공개 키 생성
- 메시지에 대한 전자 서명 생성
- 서명 검증

### 사용 방법
1. Python 환경에서 `3.1_전자서명.py` 파일을 실행합니다.
2. 프로그램이 사용자로부터 메시지를 입력받고, 해당 메시지에 대한 서명을 생성합니다.
3. 생성된 서명이 유효한지 검증합니다.

### 주요 함수
- `get_private_key()`: 개인 키를 생성하는 함수입니다.
- `euclidian(b, n)`: 주어진 두 수의 유클리드 알고리즘을 이용하여 역원을 구합니다.
- `add(point1, point2)`: 두 점을 더하는 함수입니다.
- `double_and_add(x, G)`: 주어진 점 G에 대해 x배를 계산하는 함수입니다.
- `sign(M, d)`: 메시지 M에 대한 서명을 생성합니다.
- `verify(M, S1, S2, e2)`: 서명이 유효한지 검증합니다.

### 요구 사항
- Python 3.x 이상

## 7. 비트코인 주소 생성 (4.1_bitcoin주소생성.py)

### 개요
이 프로젝트는 주어진 개인 키로부터 비트코인 주소를 생성하는 Python 스크립트입니다.

### 기능
- 비트코인 주소 생성: 사용자가 입력한 개인 키를 기반으로 비트코인 주소를 생성합니다.
- SHA-256 및 RIPEMD-160 해시 사용: 비트코인 주소 생성을 위해 필요한 해시 알고리즘을 적용합니다.

### 사용 방법
1. Python 환경에서 `4.1_bitcoin주소생성.py` 파일을 실행합니다.
2. 프로그램이 시작되면 생성할 비트코인 주소의 개인 키를 16진수 형식으로 입력합니다.
3. 입력한 개인 키로부터 비트코인 주소를 생성하고 출력합니다.

### 주요 함수
- `euclidian`: 유클리드 알고리즘을 사용하여 역원을 계산합니다.
- `add`: 타원 곡선의 두 점을 더하는 함수입니다.
- `double_and_add`: 타원 곡선의 점 곱셈을 수행하여 공개 키를 생성합니다.

### 요구 사항
- Python 3.x 이상
- `pycryptodome` 라이브러리: `pip install pycryptodome`
- `base58check` 라이브러리: `pip install base58check`

### 주의 사항
- 입력한 개인 키는 유효한 16진수 형식이어야 합니다.
- 이 스크립트는 비트코인 주소 생성 프로세스를 이해하는 데 도움을 주기 위해 작성되었습니다. 실제 비트코인 거래에 사용하기 위해서는 추가적인 보안 조치가 필요합니다.

## 8. 특정 문자열로 시작하는 비트코인 주소 (4.2_특정문자열로시작하는비트코인주소.py)

### 개요
이 프로젝트는 비트코인 주소를 생성하고, 사용자가 원하는 특정 문자열로 시작하는 비트코인 주소를 찾는 Python 스크립트입니다.

### 기능
- 비트코인 주소 생성: 임의의 개인 키로부터 비트코인 주소를 생성합니다.
- 특정 문자열로 시작하는 주소 찾기: 사용자가 입력한 문자열로 시작하는 비트코인 주소를 찾습니다.

### 사용 방법
1. Python 환경에서 `4.2_특정문자열로시작하는비트코인주소.py` 파일을 실행합니다.
2. 프로그램이 시작되면 원하는 비트코인 주소의 시작 문자열을 입력합니다. 이 문자열은 Base58 알파벳에 포함되어야 합니다.
3. 프로그램은 입력한 문자열로 시작하는 비트코인 주소를 생성하고 출력합니다.

### 주요 함수
- `get_private_key`: 안전한 임의의 개인 키를 생성합니다.
- `double_and_add`: 타원 곡선 점 곱셈을 사용하여 공개 키를 생성합니다.
- `get_bitcoin_addr`: 공개 키로부터 비트코인 주소를 생성합니다.
- `find_bitcoin_addr`: 특정 문자열로 시작하는 비트코인 주소를 찾는 메인 함수입니다.
- `is_vaild_str`: 입력된 문자열이 Base58 알파벳에 유효한지 검사합니다.

### 요구 사항
- Python 3.x 이상
- `pycryptodome` 라이브러리: `pip install pycryptodome`
- `base58check` 라이브러리: `pip install base58check`

### 주의 사항
- 입력한 문자열이 Base58 알파벳에 포함된 문자여야 합니다. (예: 숫자 1-9, 대문자 A-Z와 소문자 a-z)
- 이 스크립트는 비트코인 주소 생성 원리를 이해하는 데 도움을 주기 위해 작성되었습니다. 실제 비트코인 거래에 사용하기 위해서는 추가적인 보안 조치가 필요합니다.
